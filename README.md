# Brent Oil Price Change Point Analysis

This project analyzes structural changes in Brent crude oil prices using Bayesian change point detection to understand the impact of geopolitical and economic events.

## Project Structure
- `data/` — raw and event datasets
- `notebooks/` — EDA and analysis notebooks
- `reports/` — interim and final reports
- `requirements.txt` — Python dependencies
